## Q1.6

I originally did this as string concatenation (`output = output + lastChar + std::to_string(lastCharCount)`). The hint said don't do that, because it would be slow... which is true, it would create a new string every time and copy character-by-character. So I swapped over to `std::ostringstream`. That should make it O(n) time, O(n) space.
